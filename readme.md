## Nevers

**Criar um novo Never**
  **RF - Requisitos Funcionais**
  - O usuário deve criar um novo Never
  - O usuário não pode criar um never em um projeto que não existe
  - O usuário pode buscar um never especifico
  - O usuário pode buscar todos os nevers **Sem os projetos participados**
  - Primeiro vamos criar os projetos, se o usuário criar um nave e o projeto não existir vamos retornar um erro, mandando ele criar o projeto.
  - Após criar o projeto, quando criar o nave vamos criar também o projects_naver que vai fazer a relação dos usuários com os projetos.
